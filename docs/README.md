# Buurekrom

Zentrale Projektdokumentation. Diese Dateien sind die primäre Quelle für technische
Entscheidungen, Konventionen und Anforderungen. `PLAN.md` und `CLAUDE.md` verweisen hierher.

## Kurzbeschreibung

Buurekrom_Paths ist eine **mobile-first Web-App** zur Erfassung landwirtschaftlicher
Felder (Polygone) und Wege zu diesen Feldern (Routen) auf einer interaktiven Karte.
